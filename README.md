# XRPL Challenge 1: Trustlines for Donations

## Challenge Overview
Trustbond.app is a blockchain application that helps NGOs raise donations transparently and prove their real-world actions by recording immutable evidence on the XRPL. This increases donor trust, using features like trustlines for issued currencies, on-chain evidence for accountability, and vaults (escrows) for conditional fund release.

Inspired by Accounts & Payments (Day 1). Build a prototype using trustlines to issue "impact tokens" (IOUs) for donations. Enable donors to establish trustlines for secure transfers.

## What to Build
- Set up XRPL accounts.
- Issue a custom token (e.g., ImpactToken).
- Establish trustlines and simulate donations.
- Business: Analyze fundraising improvements (e.g., trust metrics).

## Setup
1. Clone this repo: `git clone <repo-url>`
2. Install dependencies: `npm install`
3. Run: `node index.js` (update with your wallet seeds/secrets).
